## 论文主线

CrystalTac 的问题意识是：视觉触觉传感器的“设计”与“制造”长期脱节。研究者可以设计出很复杂的 sensing mechanism（感知机制），但真正制造时要分别加工 base、lens、elastomer、marker、skin、coating，再手动装配。这个过程不仅慢，而且把性能变成工艺熟练度和装配误差的函数。

作者先把已知 VBTS 按 sensing mechanism 分类，再把每类机制对应到 contact module 的组件需求，最后提出用 PolyJet multi-material 3D printing 做 rapid monolithic manufacturing。CrystalTac 不是一个单独传感器，而是一个 family（传感器族）：C-Sight、C-Tac、C-SighTac、Vi-C-Tac、Vi-C-Sight 分别覆盖 IMM、MDM、IMM+MDM、MDM+MFM、IMM+MFM。

![[papers/images/2408.00638v1/fig_1_v5_c.png|700]]

Fig. 1 是全文的分类入口。它把 GelSight / 9DTac 放进 IMM，把 TacTip / ChromaTouch 放进 MDM，把 GelSlim / Finger Vision / TIRgel 等放进 fusion 机制。读这篇时要先理解这个谱系，否则后面的 CrystalTac family 会像是多个互不相关的原型。

对你的研究线来说，CrystalTac 的核心价值在于证明：材料、涂层、标记、透镜、皮肤、底座和装配方法不是“实现细节”，而是 sensor representation（传感器表征）的组成部分。制造一致性提高以后，才可能严肃比较不同机制或把传感器做成可复制平台。

## 1. Introduction：VBTS 为什么难从设计走到制造

Introduction 先把 VBTS 的实现拆成四步：

1. 建立 tactile features（触觉特征）和 optical features（光学特征）的对应关系，也就是 sensing mechanism。
2. 创建能实现该机制的 sensor architecture（传感器结构）。
3. 选择制造每个 sub-component（子组件）的工艺。
4. 把所有子组件装配成可用硬件。

前两步属于 design phase（设计阶段），后两步属于 creation phase（制造/创建阶段）。论文强调，传统方法把这些阶段分开，导致设计一复杂，制造就变慢、变贵、变不稳定。

作者尤其聚焦 contact module（接触模块），因为 illumination module（照明）和 vision module（相机）通常可买现成电子件，而接触模块要同时满足柔性、透明、遮光、标记、涂层、耐磨、成像和装配。一个典型 VBTS 接触模块包含六类组件：

- base（基座/外壳/框架）
- lens（透明支撑/光学窗口）
- elastomer（弹性体）
- marker（标记）
- skin（外层皮肤/遮光/保护层）
- coating（功能涂层）

论文提出的 CrystalTac family 不是为了宣称某个型号性能最强，而是为了证明 monolithic manufacturing 可以适配多种 sensing mechanisms，从而降低 VBTS 的 design complexity、process complexity、time complexity 和 quality variability。

## 2. Background：机制、组件和装配的系统分类

### 2.1 Typical Sensing Mechanism of Known VBTSs

#### 2.1.1 IMM：Intensity Mapping Method

IMM 用 pixel intensity values（像素强度值）来表示 tactile features。典型例子包括：

- GelSight-type sensors：用 reflective coating 和 RGB lighting，把颜色/亮度变化映射到 surface normal 或 depth。
- DTac / 9DTac：用 translucent gel + opaque layer，让 darkness（暗度）对应深度或力/扭矩。
- depth camera based VBTS：像素值直接包含距离信息。

IMM 的优点是充分利用相机的高空间分辨率，适合 continuous geometry / texture（连续几何和纹理）。缺点是对光路、涂层、透明度、反射模型和遮光结构高度敏感。

#### 2.1.2 MDM：Marker Displacement Method

MDM 把接触信息映射成 marker pattern 的运动或形变。论文进一步提到 2D-MDM、2.5D-MDM 和 3D-MDM：

- 2D-MDM：单相机观察 marker 在水平/垂直方向移动。
- 2.5D-MDM：marker 的大小或形状变化间接反映深度。
- 3D-MDM：用 stereo vision 或多层 marker 精确估计 marker 深度。

TacTip 的 pin-shaped markers、GelForce 的双层 marker array、ChromaTouch 的颜色滤片、DenseTact 的 dense random color pattern、Tac3D 的 CMP 都属于这个谱系。MDM 的重点不是亮度本身，而是 marker 如何在软体形变中编码力、剪切、深度或姿态。

#### 2.1.3 MFM：Modality Fusion Method

MFM 将 tactile input 与 visual / proximity sensing 结合。它不能单独作为触觉机制，通常要和 IMM 或 MDM 结合。例如 see-through sensors 可以在未接触或弱接触时保留外部视觉，在接触区域同时产生触觉亮度变化。

#### 2.1.4 Multi-Mechanism Fusion

多机制融合是当前 VBTS 很重要的趋势：

- **IMM + MDM**：GelSlim、GelSight Wedge、DenseTact、UVtac 等，在 reflective / intensity mapping 基础上加 marker，用于 shear / force / slip。
- **MDM + MFM**：Finger Vision、ViTacTip、SpecTac 等，用透明皮肤和 marker 同时获得视觉与触觉。
- **IMM + MFM**：STS、StereoTac、VisTac、TIRgel 等，用透明/半透明 coating、TIR 或 binocular vision 实现视觉-触觉切换。

这组分类帮助你把后续论文放在同一张图里：GelSight 是 IMM，GelSlim 是 IMM+MDM，TacShade 更接近 MDM 与 greyscale/intensity 的混合，CrystalTac 是制造方法对这些机制的统一承载。

### 2.2 Typical Manufacturing Method of Known VBTSs

![[papers/images/2408.00638v1/fig_2.png|700]]

Fig. 2 把 VBTS 的代表性属性拆成 base、lens、elastomer、marker、skin、coating 和 assembly。这里的关键信息是：不同机制会要求不同材料属性，制造方法不是可替换小事。

原文 Table 1 很长，可以压缩成下面的比较：

| 组件 | 常见制造方法 | 技术含义 |
| --- | --- | --- |
| Base | 3D printing、mould-forming | 决定刚性、安装、紧凑性和外部形态。 |
| Lens | laser-cut acrylic、mould-forming、commercial lens | 决定成像透明度、内部支撑和复杂曲面可制造性。 |
| Elastomer | mould-formed silicone、commercial gel、air/latex membrane | 决定柔软度、透明度、接触变形和寿命。 |
| Marker | ink printing、film sticking、embedded beads、integral printed pins | 决定 marker 位移/形变如何编码触觉信息。 |
| Skin | silicone cast/spray、fabric/film、printed body | 决定遮光、保护、摩擦和接触界面。 |
| Coating | aluminum/bronze flakes、airbrush、paint dipping、mirror spray | 决定 IMM 的反射/透明/可控光学属性。 |

论文特别指出 coating layers 和 skin 不同：skin 偏保护和遮光，coating 偏把 tactile features 转成 optical imaging。GelSight 的金属粉末 reflective coating、STS 的 mirror spray controllable coating 都属于后者。

### 2.3 Assembly Method

装配维度分三类：

- **Assembly tool（装配工具）**：manual assembly（手工装配）仍然主流；machine assembly（机器/打印机装配）更一致但依赖设备。
- **Assembly workflow（装配流程）**：serial assembly（串行装配）多见，parallel assembly（并行装配）效率高但不适合复杂结构。
- **Assembly mechanism（装配机制）**：physical assembly（结构卡合）便于模块化；chemical assembly（胶水/材料粘接）提高耐久与稳定。

CrystalTac 的目标就是尽量把这些装配动作合并到打印过程里，从而减少人工误差。

## 3. Design and Creation：CrystalTac family 怎样被制造出来

### 3.1 Manufacturing Feasibility

![[papers/images/2408.00638v1/fig_3_v2.png|700]]

Fig. 3 是一体化制造的核心图：设计好 CAD 和材料分配后，PolyJet 打印同时完成多组件结构，最后只需去除 support materials。它的优势有两个：

- **High printing quality（高打印质量）**：喷射液滴小，XY 和 Z 方向都有较高空间分辨率，适合 marker、skin、lens 等细结构。
- **Multi-material printing（多材料打印）**：多个 print heads 可以混合 Vero、Agilus30、DraftGrey、support 等材料，调控透明度、颜色和硬度。

但这不是无限制的。论文明确说，当前打印材料缺少 metal powder，因此依赖 reflective coating 或 controllable coating 的某些 IMM-GelSight 方案还不能完全一体化打印。这一点很重要：CrystalTac 不是说所有 VBTS 都已经能完美打印，而是说多数 contact module structure 可以被一体化制造显著简化。

实际可行性由三件事决定：

1. **Suitable sensor design**：结构必须真的能实现目标机制。比如 MDM 需要 marker 在正确位置，MFM 需要视线路径尽量少遮挡，IMM 可能需要多层堆叠。
2. **Combination setup of print material**：打印机材料槽有限，最少要有 Agilus30 Clear、Vero White/Black 和 support；有空余 print heads 时优先加载 Agilus30 White/Black 或 VeroClear。
3. **Assignment setup of print material**：材料要按组件需求分配，例如 MFM 需要透明低畸变 skin，MDM 需要有效遮光和可见 marker。

论文给出两个经验范围：

- printed lens（打印透镜）厚度建议 **2-3 mm**，平衡结构强度和成像质量。
- printed elastomer（打印弹性体）高度建议 **2-5 mm**，平衡透明成像和柔软度。

| 分支 | 对应机制 | 结构要点 | 读法 |
| --- | --- | --- | --- |
| C-Sight | IMM | 黑色 skin、translucent layer、white layer、clear elastomer | 借鉴 DTac / 9DTac，让黑白层距离变化造成 pixel intensity change。 |

## 4. Performance Evaluation：功能、成本和自定义能力

### 4.1 Object Recognition with C-Tac

C-Tac 使用 single-layer dot markers 和黑色 skin。实验物体包括 dot、ring、sphere、curve、waves、multiple dots 六个打印件。接触后 marker 的位移和形变可以区分不同物体：例如 hollow ring 主要移动外围 markers，sphere 则让中心 markers 位移最大。

![[papers/images/2408.00638v1/fig_5_c.png|700]]

作者采集 1200 张图像，每类 200 张，用 DenseNet121 做分类，测试准确率为 **98.3471%**，测试集中 121 张对了 119 张。这个实验验证的是 MDM-CrystalTac 能把物体几何转成可学习的 marker distribution。

### 4.2 Object and Texture Hybrid Recognition with Vi-C-Tac

Vi-C-Tac 用 double-layer dot markers 和透明 skin，同时看 tactile marker 和外部 fabric texture。实验用三种 fabric（blue cotton、glossy chemical fibre、rough hemp）包住三种打印件（curve、waves、dots），形成 9 种排列。外部视觉可以看 fabric texture，但看不到下面的打印形状；marker 则提供隐藏物体形状线索。

每种排列采 200 张，共 1800 张。DenseNet121 后接两个 decoupled classifier heads，分别做 object recognition 和 texture recognition。结果为：

| 任务 | 准确率 |
| --- | ---: |
| object recognition | 100% (182/182) |
| texture recognition | 99.45% (181/182) |

这个实验是 MDM + MFM 的直接证据：透明皮肤保留了 visual texture，双层 marker 捕捉了 tactile shape。

### 4.3 See-through-Skin Exploration with Vi-C-Sight

Vi-C-Sight 使用纯 Agilus30 Clear 作为 elastomer，像 light-conductive plate 一样依赖 total internal reflection（全内反射）形成 proximity-tactile fusion。未接触区域保持透明，接触区域因为局部变形和光路变化出现亮度差异。

![[papers/images/2408.00638v1/fig_6_c_v2.png|700]]

Fig. 6A 展示了 fabric fibers、fingerprints、embroidery deformation 等细节能被映射到像素强度变化。Fig. 6A.b 用带 human figure 的 embossed base 和 skeleton fabric 做探索，说明 Vi-C-Sight 可以同时观察表面纹理和底下的几何凸起。

这一节没有给统一分类指标，更多是 qualitative proof-of-concept。它的价值在于展示 see-through skin 不只是透明相机，而是 tactile brightness change 与 visual information 的耦合。

### 4.5 Customised Flexibility

![[papers/images/2408.00638v1/fig_7.png|700]]

Fig. 7 将自定义能力分成三类：

- **Internal structure（内部结构）**：可以用 Agilus30 Black 遮光，也可以用柔性 2.5D markers 增加形变线索，或用刚性 marker 保持几何。

## Supplementary Materials：子组件优化和 marker 设计

### 5.1 Lens Manufacturing：打印透镜的光学畸变

![[papers/images/2408.00638v1/supplymentary_lens_evaluation.png|700]]

PolyJet 打印的 VeroClear / VeroUltraClear 透明度较好，但打印表面会出现 strip-shaped textures（条纹纹理）。这些纹理使反射和折射不再规则，造成图像畸变。论文用 Snell's law（斯涅尔定律）解释折射关系：

$$
\frac{\sin \theta}{\sin \theta_l}=\frac{N_{lens}}{N_{air}}
$$

变量含义：

- $\theta$ 是入射角。
- $\theta_l$ 是折射角。
- $N_{lens}$ 和 $N_{air}$ 是透镜材料和空气的 refractive index（折射率）。

因为打印表面法线在局部变化，平行光经过表面后不再保持平行。不同于 barrel / pincushion / mustache distortion，这种随机表面纹理导致的畸变不能简单用棋盘格相机标定参数校正。

作者测试两种优化：

- **Adjusting printing thickness**：1/2/3/5 mm 打印透镜与 acrylic lens 对比。低于 3 mm 时与 acrylic 的 similarity 约 86%，5 mm 降至 83%；1 mm 最高约 87.5%，但结构强度可能不足。
- **Applying acrylic spray**：喷涂可以填平表面纹理，降低 X/Y 方向差异。喷涂后对真实 pattern 的 similarity 最高约 86%，在 3 mm 厚度时较好。

综合结论：printed lens 建议 **2-3 mm**，再结合 acrylic spray 改善表面 finish。

### 5.1 Elastomer Manufacturing：透明度和柔软度的取舍

![[papers/images/2408.00638v1/supplymentary_elastomer_lens_c.png|700]]

Agilus30 Clear 透明且柔性，但 Shore hardness 约 30A，对某些 GelSight 类软弹性体来说偏硬。作者采用 multi-layer grid structure 降低有效刚度，并测试 2/5/10/15/20 mm 高度的 elastomer。

结果显示高度增加会造成两类问题：

- tone（色调）变暗偏黄，主要因为内部 support material 透光率低于 Agilus30。
- pattern details（图案细节）逐渐丢失，SSIM similarity 从 2 mm 高度的约 76% 降到超过 10 mm 时的约 52%。

但高度太低也会降低柔软度，因为 grid 层数少，支撑芯的可变形范围有限。综合结论是 printed elastomer 高度建议 **2-5 mm**：薄层透明度好，厚层柔软度更好。

### 5.2 Marker Design

![[papers/images/2408.00638v1/supplymentary_marker_pattern.png|700]]

Supplementary Fig. 10 提出四类 marker：

- **Dot marker**：最常见，可用 rigid Vero 做 2D marker，也可用 Agilus30 / Tango 做可形变 2.5D marker。
- **Double-layer marker**：利用软体变形沿纵向递减的特性，通过上下层 marker 差异增强 shear / dynamic tactile features。
- **Voronoi marker**：连续 tessellation pattern，比点阵有更高空间连续性，接近 continuous marker pattern（CMP）的思想。
- **Coordinate marker**：像 gyroscope-like 3D pointer，每个 marker 单独编码局部 6D pose，适合 force estimation 或 dynamic motion tracking。作者认为这种 stereo marker 几乎只有 monolithic manufacturing 才容易实现。

这一节对你的阅读线很有价值：marker 不只是“点越多越好”，marker 的材料、层数、几何和颜色梯度都会改变触觉表征。

## 与其它文献的关系

- 与 GelSight / GelSlim / GelSight Wedge 相比，CrystalTac 不主要追求某个 sensor 的最佳几何重建，而是把 VBTS 的结构组件和制造流程系统化。
- 与 TacTip / TacShade 相比，CrystalTac 对 MDM 的贡献在于 marker 可以被材料和空间结构一起设计，而不是只在软皮肤上放点。
- 与 PBR Design 相比，CrystalTac 解决的是 manufacturing and assembly（制造与装配）问题；PBR Design 解决的是 optical design simulation（光学设计仿真）问题。两者可以互补：先用仿真设计机制，再用一体化打印稳定制造。
- 与 AllSight 相比，AllSight 是一个具体可迁移圆形传感器；CrystalTac 是多个机制可制造性的统一平台。

## 局限与可追问点

1. **不是性能竞赛型论文。** C-Tac / Vi-C-Tac 的分类准确率很高，但数据规模和任务较简单，不能直接说明它们超过最佳 VBTS。

2. **金属反射涂层仍是材料短板。** 由于 PolyJet 材料不含 metal powder，传统 GelSight 式 reflective coating 无法完全用当前一体化打印替代。

3. **Vi-C-Sight 的 see-through 实验偏定性。** 它展示了透明触觉的潜力，但没有给统一的 depth / force / pose 指标。

4. **长期 durability 尚未系统评估。** 一体化打印能提高一致性，但材料老化、透明度变化、skin 磨损、support 去除残留都可能影响长期信号。

5. **成本估计依赖特定打印设备和材料价格。** Table 2 的 GBP 成本不含 VAT，并且未计入设备折旧、失败率、后处理和电子模块。

6. **设计自由度仍受 print heads 和材料槽限制。** 多机制融合需要更多材料，打印机材料组合固定时可能限制方案。

## 一句话定位

CrystalTac 是一篇“把视觉触觉传感器从单个硬件原型推进到机制-组件-制造平台”的论文。它的核心贡献不是某个传感器在某个 benchmark 上最好，而是提供了一个可复用的框架：先按 IMM / MDM / MFM 理解感知机制，再按 base / lens / elastomer / marker / skin / coating 分解制造需求，最后用 multi-material monolithic manufacturing 让复杂 VBTS 变得可复制、低成本且可定制。
