## 论文主线

这篇论文有两条互相连接的主线。第一条是改进 Fingertip GelSight 的几何测量：旧版本能给出高分辨率图像，但使用 semi-specular membrane 和非均匀 illumination，导致 RGB 到 surface normal 的映射不够准确。作者改用 Lambertian membrane 和新的三色 LED 照明系统，让颜色变化在更大的法线角度范围内保持线性，从而提升 surface normal 和 depth map 的精度。

第二条是把改进后的 GelSight 用于 slip detection（滑移检测）。传统滑移检测常看力比值、振动或加速度，而这篇利用 GelSight 图像里的两类信息：object texture（物体纹理）和 membrane marker motion（传感器膜面标记运动）。如果物体没有滑移，纹理和 marker 应该同步运动；如果发生滑移，二者会出现相对平移或相对旋转。

这篇在 GelSight 系列中很关键：它把“看清局部 3D 几何”和“用触觉图像闭环改善抓取稳定性”放到同一个小型 gripper sensor 里。后续 GelSlim 进一步处理紧凑集成和耐久标定，AllSight / DenseTact 进一步处理曲面传感器和完整状态估计，PBR Design 则把这里靠经验调的光路设计推进到仿真优化。

## I. Introduction：从高分辨率图像到可靠抓取

Introduction 先说明 tactile sensing 对抓取的重要性：机器人需要判断是否接触、夹持力是否足够、物体是否开始滑动。Slip（滑移）是常见抓取失败形式，人类可以通过皮肤拉伸和微小振动察觉，机器人则需要可工程化的触觉信号。

已有 Fingertip GelSight 已能达到高空间分辨率，原文提到 640 x 480 pixels、约 0.024 mm/pixel，并已用于 USB insertion 等任务。但旧设计有两个问题：

- surface normal measurement（表面法线测量）不准，导致 3D map 粗糙；
- fabrication（制造）难，需要手工精确调整组件。

本文的新设计希望同时解决这两个问题：让 illumination 更均匀、让反射膜更适合 photometric stereo，并让框架和照明系统更容易 3D 打印和装配。

## II. Related Work：光学触觉、GelSight 和滑移检测

### II.A Optical Tactile Sensors

光学触觉传感器用相机观察接触表面形变，因此能用低线缆复杂度获得高空间分辨率。早期方法包括 optical waveguide、pyramidal projection rubber、marker flow field 等。它们可以检测接触点、二维形状、力或粗略 3D 形状，但通常空间分辨率受 sparse texture / markers 限制，难以恢复高精度 3D map。

### II.B GelSight Sensor

GelSight 的核心是 soft silicone gel + reflective membrane + colored LEDs + camera。三色光从不同方向照射，接触后膜面局部法线改变，相机颜色变化就可映射到 surface normal。原始 GelSight 能捕捉 microgeometry（微几何），但早期指尖版为了小型化牺牲了照明均匀性和几何精度。

旧 Fingertip GelSight 的问题可以理解为两个映射噪声：

- 非均匀照明让同一个 surface normal 在不同位置产生不同颜色。
- semi-specular surface 让颜色变化不是单调线性，某些颜色可能对应多个不同斜率。

### II.C Slip Detection

传统滑移检测常用 shear-to-normal force ratio（切向/法向力比）、vibration（振动）、acceleration（加速度）或接触力频率变化。GelSight 的优势是它直接看到接触纹理和膜面形变，因此可以从 slip 的定义出发：slip 是 object 与 finger 之间的 relative displacement（相对位移）。

这为本文的滑移检测提供了更一般的信号：不必预先知道物体重量、几何、材质或粗糙度，只要能追踪接触区域的纹理和 marker 运动，就能判断是否出现相对运动。

## III. Design and Fabrication of New GelSight Sensor

新设计围绕四个目标：用 Lambertian surface 替代 semi-specular surface，优化 sensing surface 的照明均匀性，使用更标准化的 3D printable framework，并保持夹爪可安装的紧凑结构。

### LED 和照明结构

作者使用带小型 collimating lens 的 Osram SMD LEDs，镜头视角为 30 度。LED 以 2 x 4 array 排列，并用红、绿、蓝三组 LED 阵列从三个方向照明。LED mounts 倾斜约 71 度，让光覆盖整个 sensing surface。

这个角度设计有两个作用：

- 旋转对称结构让中心区域 R/G/B 强度更均衡。
- 大倾斜角让不同 surface normal 产生更可区分的颜色变化，有利于法线估计。

### Tray、膜层和支撑

传感器框架是透明/半透明 hexagonal plastic tray，由 Formlabs clear resin 3D 打印。半透明 tray 可以 homogenize（均匀化）LED 光，同时保持较高透光。

关键材料变化是把旧的 semi-specular coating 换成 aluminum powder mixed in silicone base 的 Lambertian coating。Lambertian membrane 牺牲了一些强反光对比，但让 photometric stereo 的物理假设更接近成立。

支撑结构使用 laser-cut transparent hexagonal acrylic window，并用同类 silicone 填充 hollow tray，以匹配 gel 的 refractive index（折射率），减少内部反射和成像 artifact。

### 标定 RGB 到法线的 LUT

在进行 3D reconstruction 前，需要标定 lookup table：

1. 用直径 3.96 mm 的球压入 elastomer gel。
2. 根据球的已知几何计算每个像素的 surface normal。
3. 记录球压入引起的 RGB color value change。
4. 自动生成 RGB 到 surface normal 的查找表。
5. 在不同位置重复标定，并对多个 LUT 求平均，以降低 spatial illumination variance。

这个 LUT 思路和你关心的 Global LUT / SA-LUT 很直接相关：如果照明空间非均匀，单一全局 RGB-to-normal mapping 就会出错；本文通过硬件光路和多位置标定同时缓解这个问题。

## IV. Evaluation of Geometry Measurement

### IV.A Gradient vs. Color Change

![[papers/images/1708.00922v1/ColorDiff_v4.jpg|700]]

Fig. 4 比较旧传感器和新传感器在 calibration sphere 上的颜色变化。旧传感器在 pitch angle 约 5-25 度内近似线性，但超过 30 度后颜色变化不再单调，出现一个 color value 对应多个 surface normal 的 ambiguity（歧义）。这种歧义会让 RGB 到法线的反演失效。

新传感器的 color change 更平滑，虽然变化幅度不一定更强，但在 5-60 度范围内保持更好线性。这里的取舍很重要：对几何重建来说，可逆、单调、稳定的映射比“颜色变化很强”更有价值。

### IV.B Mapping Accuracy of the Lookup Table

![[papers/images/1708.00922v1/evaluation_2_v3.jpg|700]]

Fig. 5 比较 measured surface normal angles 和 ground truth。旧传感器的 pitch angle $R^2$ 为 **0.557**，新传感器提升到 **0.818**。Yaw angle 两者都较好，但新传感器仍略优。

这说明新照明系统最主要改善的是 pitch angle（与表面倾斜程度相关）的可测性。对深度重建来说，pitch 误差会直接影响梯度积分后的 height map。

### IV.C Spatial Difference of Illumination

Fig. 6 用不同位置 calibration images 的 $R^2$ distribution 来衡量空间照明差异。新传感器的 pitch/yaw $R^2$ 分布整体更高，并且没有旧传感器那种低 $R^2$ 长尾。作者把这解释为 sensing surface 上没有明显 blind spot。

这部分结论可以写成：新设计不只是中心点精度变好，而是整个接触面上的法线估计更一致。

### IV.D Quality of 3D Shape Reconstruction

论文用 ball array、human finger、watch chain 和 quarter coin 做定性重建。新传感器的 depth map 更平滑，ball array 能保持球形而不是锥形噪声，quarter coin 上 eagle feather 的细节更清楚。

这组实验没有给统一数值表，但视觉上支撑了前面 $R^2$ 指标：颜色到法线更稳定，积分出的深度图也更少噪声和空间失真。

## V. Slip Detection with New GelSight Sensor

作者把 slip detection 分成三种线索：

1. object texture 与 marker 的 relative displacement；
2. marker motion distribution，特别是外围 marker 与中心 marker 的运动差异；
3. contact area decrease，用于判断严重滑移。

### 线索一：物体纹理和 marker 的相对运动

![[papers/images/1708.00922v1/MarkerSlip.png|700]]

Fig. 7 展示对有明显纹理的物体，取一个 120 x 120 window，计算相邻帧之间 object texture 和 markers 的 translation / rotation。如果二者差异超过阈值，或累积相对平移/旋转过大，就认为 slip 正在发生。

直观解释：无滑移时，物体纹理和传感器膜面 marker 在相机图像中应当一起移动；有滑移时，物体纹理在接触面上“擦过去”，与 marker 的运动不同步。

### 线索二：marker motion distribution

对光滑曲面物体，纹理可能不足，例如圆柱瓶或金属罐。此时作者使用 Yuan et al. 2015 的观察：incipient slip（初始滑移）从接触外围开始，外围区域的 marker 位移会小于中心区域。

论文定义比值：

$$
r=\frac{\|v_p\|_{max}}{\|v_i\|_{max}}
$$

变量含义：

- $v_i$ 是接触区域所有 markers 的运动向量。
- $\|v_i\|_{max}$ 是这些 marker 运动的最大范数，通常来自接触中心区域。
- $v_p$ 是 approximate peripheral area markers（近似外围 marker）的运动向量。
- $\|v_p\|_{max}$ 是外围候选 markers 中最大运动范数。

如果 $r$ 低于阈值，说明外围 marker 相对中心 marker 运动不足，接触可能已经开始从边缘滑动。论文实验中阈值选 **0.8**。

![[papers/images/1708.00922v1/MarkerSlip1.png|700]]

Fig. 8 展示如何选择外围 marker：对每个 marker 附近区域计算 RGB intensity change，选择变化最大的 8 个 marker 作为接触边界候选，再从中取最大运动向量。

### 线索三：contact area decrease

当接触面积显著下降时，通常说明严重 slip 已经发生。作者用当前图像和无接触图像的 color intensity difference 来检测 contact area。这种方法简单，适合实时闭环抓取，但对非常光滑、低剪切力和纹理不足的接触会有局限。

## VI. Experimental Result on Robotic Slip Detection

### VI.A Experimental Setup

机器人系统由 UR5 6DOF arm、WSG-50 parallel gripper 和新 GelSight sensor 组成。UR5 reach 为 850 mm，repeatability 为 +/-0.1 mm。WSG-50 原始最大 opening width 为 110 mm，装上传感器后为 80 mm，最小 closing speed 为 5 mm/s。

实验对象是 37 个日常物体，尺寸、形状、材料和表面纹理都不同。抓取流程是：夹爪慢慢接近物体，GelSight signal 判断是否接触以及 force 是否足够；抓住后机器人慢慢抬起 3 cm，并在抬起过程中检测 slip。

### VI.B Slip Prediction

每个物体抓 7-10 次，用不同 contact threshold 形成不同夹持力。人眼观察视频，给每次实验分三类：

- successful cases：物体和夹爪相对运动很小，抓取稳定。
- slip cases / failure cases：出现明显 translational 或 rotational slip，抓取失败。
- border cases：勉强抬起，但很容易掉，或平衡前已经发生滑移；这一类作者认为判断 slip 或 no slip 都可接受，但记录 no-slip 比例。

Table I 的结果：

| Case type | Sample number | Correct measure |
| --- | ---: | ---: |
| successful cases | 147 | 79% |
| slip cases | 116 | 84% |
| border cases | 52 | 60% no-slip measure |

失败原因也值得记住：

- 在光滑平面、夹持力很小的情况下，物体可能沿传感器表面直接滑动。剪切力小到 marker motion 被噪声淹没，同时纹理不足，导致 slip cases 中约 **28%** 的失败。
- 对 flat object 发生明显 rotational slip 时，简化的 marker norm 方法可能失败；这也造成 slip cases 中约 **28%** 的失败。作者认为更完整的 marker rotational movement 计算可以改善。

### VI.C Grasp Control with Slip Detection

闭环抓取实验使用 33 个可被抬起的物体，共 99 次抓取。如果抬起过程中检测到 slip，机器人停止、放下物体、释放，然后用更高 contact threshold 重新抓取。每次重新尝试 threshold 增加 1.2 倍。

结果：99 次中成功稳定抓取 **88 次**，成功率 **89%**。每次实验目标物体被抓取 1-6 次，平均 **2.3 次**。失败主要发生在夹持力太小、接触信息不足、滑移信号无法可靠估计的场景。

## 与其它文献的关系

- 与 Yuan 2017 GelSight 综述/系统论文相比，这篇更像一个面向 robot gripper 的改进版：重点是小型化、精度提升和 slip closed-loop。
- 与 GelSlim 相比，这篇先解决几何精度和滑移检测；GelSlim 后续更强调 slim finger、长期耐久和 calibration over usage。
- 与 AllSight 相比，这篇仍是较平面/局部的 GelSight finger；AllSight 进一步走向 round 3D sensing surface 和 zero-shot state estimation。
- 与 PBR Design 相比，这篇的照明和膜层设计仍是经验优化；PBR Design 试图把 illumination、shape、coating 的选择变成可仿真优化问题。

## 局限与可追问点

1. **滑移检测仍有手工阈值。** 例如 $r<0.8$ 是实验中选择的阈值，复杂物体、不同速度或不同材料下可能需要调整。

2. **对光滑低纹理物体较弱。** 如果既没有明显 object texture，又只有很小 shear signal，texture tracking 和 marker motion 都可能不稳定。

3. **旋转滑移处理被简化。** 作者为了实时性没有完整估计 marker flow center，只用运动向量范数近似，导致 flat object 的 rotational slip 失败。

4. **3D 重建是定性展示多于系统 benchmark。** 几何部分给出 $R^2$ 改善，但 depth reconstruction 没有大规模数值表。

5. **仍依赖单传感器标定。** LUT 需要用球压入标定，并在多个位置平均。跨传感器、长期老化和批量一致性不是本文重点。

6. **膜层和照明仍是脆弱变量。** Lambertian coating、LED 对齐、tray 透明度、支撑折射率匹配都会影响 RGB-to-normal mapping。

## 一句话定位

Improved GelSight 是 GelSight 指尖从“高分辨率触觉图像”走向“更准几何 + 可用于抓取闭环滑移检测”的关键论文。它证明光路、反射膜和标定能显著改善 RGB-to-normal 映射，也证明同一触觉图像里可以同时读出局部几何、marker 剪切和接触稳定性。
