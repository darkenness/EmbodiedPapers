## 论文主线

TransTac 关注的不是“再做一个更高分辨率的触觉皮肤”，而是一个更具体的断点：机器人手靠近物体时，RGB-D / stereo vision（视觉深度）在极近距离会失效；真正接触后，传统 GelSight-style 或 DTact-style VBTS 又通常依赖 opaque coating（不透明涂层），只能看见接触形变，不能继续观察物体外观和非接触凹陷几何。于是，从 pre-contact observation（接触前观察）到 physical contact（物理接触）之间存在一个 geometric sensing gap（几何感知空隙）。

![[papers/images/2606.04477v1/Motivations.jpg|700]]

**Fig. 1: Motivation of TransTac.** 图中把这个 gap 分成两侧：RGB-D camera 能给 global depth（全局深度），但近距离可靠性下降；coated VBTS 能重建 contact deformation（接触形变），但不透明涂层挡住视觉。TransTac 的目标是把 binocular vision（双目视觉）和 UV-encoded transparent tactile sensing（紫外编码透明触觉感知）放进统一硬件结构。

本文的核心回答是：使用 transparent silicone elastomer（透明硅胶弹性体）保留视觉穿透性，在弹性体内嵌入 UV-reflective semitransparent markers（紫外反光半透明标记），再用双目相机同时服务视觉观察和标记三角化。白光照明用于 RGB 观察，365 nm 紫外照明用于凸显 markers；算法侧用 lightweight marker detector（轻量标记检测器）、ByteTrack、prior-guided Delaunay stereo matching 和 RGB-D fusion with scale optimization 把 marker deformation（标记形变）转成近接触几何。

因此，TransTac 的位置介于三条文献线之间：一条是 GelSight / DTact 的 contact geometry reconstruction（接触几何重建），一条是 FingerVision / TacTip / SpecTac 的 marker-based tactile reconstruction（基于标记的触觉重建），另一条是 RGB-D / stereo depth 的 near-contact perception（近接触视觉深度）。它的主张是：透明材料和 UV 编码让同一设备在视觉和触觉之间“过渡”，而不是把视觉传感器和触觉传感器并排堆起来。

## 摘要与核心贡献

摘要先把问题讲成一个互补失败：VBTS 能恢复 high-resolution contact geometry（高分辨率接触几何），但多靠 opaque elastomer layers（不透明弹性体层）；RGB-D cameras 能提供 global depth perception（全局深度感知），但 near range（近距离）性能明显下降。TransTac 通过 transparent ultraviolet-encoded binocular VBTS（透明紫外编码双目视觉触觉传感器）把 visual observation（视觉观察）和 marker-based tactile reconstruction（基于标记的触觉重建）合在一个 compact device（紧凑设备）里。

核心技术由三部分组成：

- 材料与硬件：transparent elastomer embedded with UV-reflective markers（嵌有紫外反光标记的透明弹性体）。
- 视觉/触觉分离：white LED 用于 RGB 观察，365 nm UV illumination 用于 marker detection。
- 几何算法：prior-guided Delaunay stereo matching 支持 sparse triangulation，再与 dense RGB-D estimate 对齐。

摘要给出的关键数字包括：Delaunay matching 比 global assignment baseline 的 correspondence robustness（对应鲁棒性）提升约 21%；TransTac tactile images 在 zero-shot recognition（零样本识别）中最高 83.3%；DINOv2 class-center similarity 从约 0.2 提升到 0.774 以上；硬件成本约 70 美元。

## I. Introduction / 研究问题

Introduction 的逻辑很直接：机器人操作需要触觉，因为视觉在遮挡、光照变化和接触界面附近不可靠；但现有触觉传感器又常牺牲视觉穿透性。作者指出，很多 VBTS 是 geometry-oriented（几何导向）的，例如 GelSight、DIGIT、DTact、DenseTact 等，它们擅长接触形变重建，却通常用反射膜或涂层把外部视觉遮掉。

本文要解决的核心 challenge 是：如何在 pre-contact observation 和 physical contact 之间保持 reliable geometric perception（可靠几何感知）。RGB-D 相机在接近最小测距距离时有效深度快速下降；coated VBTS 只有接触后局部形变，不能恢复 recessed / non-contact geometry（凹陷或非接触几何）。如果机器人在抓取前后都要知道物体形状，那么这两类设备之间存在断层。

TransTac 的入手点是透明化与编码化。透明弹性体保留外部 visual clarity（视觉清晰度），UV markers 在紫外光下提供可跟踪几何锚点。双目相机提供 stereo geometry，使 marker correspondences（标记对应）可以被三角化为公制深度。这样，传感器既能在接触前看物体，又能在接触时通过标记位移重建局部接触。

这一节的三个贡献分别对应硬件、检测、匹配：

- 硬件贡献：透明紫外编码双目 VBTS。
- 检测贡献：面向密集半透明 markers 的轻量模型。
- 算法贡献：先验引导 Delaunay 双目匹配，用极线约束和局部几何结构提高对应可靠性。

## II. Related Work / 相关工作与定位

### A. Vision-Based Tactile Sensing

VBTS 通过相机观察 elastomer deformation（弹性体形变）来恢复接触信息。GelSight / GelSlim 一类系统使用 reflective coatings（反射涂层）和 photometric stereo（光度立体）做高分辨率表面几何；DIGIT、DTact 等使用学习或灰度变化映射来做接触形状推断。这些方法的共同优势是 contact-local geometry（接触局部几何）强，问题是视觉路径封闭，不能透过触觉面继续观察外部场景。

作者也承认 transparent or vision-through tactile sensors（透明或可穿透视觉的触觉传感器）已有探索，例如透明 optical tactile skin。但难点在于：透明材料让视觉能通过，却会削弱传统反射涂层/受控照明带来的几何重建稳定性。TransTac 的特殊点是用 UV markers 把“透明视觉”和“可跟踪触觉几何”分开编码。

### B. Marker-Based Tactile Reconstruction

marker-based tactile reconstruction 的思路是：不直接从整幅图像光照反推深度，而是跟踪 membrane（膜层）或 elastomer 内部 markers 的位移。FingerVision、TacTip、TAC2Pose 等系统都属于这条线。透明实现和 UV 分离也有先例，例如 SpecTac 使用 UV illumination 激活 markers。

TransTac 的挑战在于 markers 是 dense semitransparent markers（密集半透明标记），接触后会形变、遮挡、亮度变化，而且左右相机看到的 marker 集合不完全一致。因此，仅靠 blob detection（斑点检测）或传统 optical flow（光流）容易漏检、漂移或交换身份。

### C. Stereo Correspondence and Near-Contact Depth Sensing

双目匹配通常依赖 epipolar constraint（极线约束）把搜索限制到一维 scanline（扫描线）。传统方法包括 Hungarian matching（匈牙利全局分配）和 ELAS 等支撑点传播算法。但在密集 marker 图案中，很多点外观相似，且弹性体变形会改变局部几何，因此全局分配可能把重复点匹配错。

另一方面，RGB-D sensing（RGB-D 深度感知）在 near-contact scenarios（近接触场景）中受到最小测距和三角化几何约束影响，容易产生 missing depth（缺失深度）或 scale error（尺度误差）。所以本文把 stereo marker triangulation 作为近接触可靠锚点，再对 dense depth model 做尺度修正。

## III. Design and Fabrication / 方法与系统

![[papers/images/2606.04477v1/flowchart_2.jpg|700]]

**Fig. 2: Overview of the TransTac framework.** 图中把 TransTac 分成两条并行但互相补充的链路：RGB/white-light 图像用于视觉观察和 dense depth estimation；UV 图像用于 marker detection、stereo correspondence 和 triangulated sparse depth。最后 sparse marker depth 会对 dense depth 做 scale alignment。

### A. TransTac Design Principles

设计原则是 compact, low-cost stereo configuration（紧凑低成本双目结构）加 transparent silicone membrane（透明硅胶膜）。透明膜让外部物体外观能通过触觉界面被相机观察；嵌入 UV-reflective markers 让同一界面在紫外光下变成可检测的 marker field（标记场）。

time-multiplexed illumination 是这里的关键工程取舍。白光/RGB 模式服务外观与视觉深度；UV 模式服务 marker 定位。它避免了额外光学分束或复杂滤光结构，但会带来 illumination switching latency（照明切换延迟），这也是作者在局限里承认的问题。

### B. Sensor Hardware and Fabrication

硬件由以下组件组成：

| 模块 | 具体实现 | 作用 |
| --- | --- | --- |
| stereo cameras | HBVCAM-2436-R V11 USB camera modules，水平 FOV 100 度，manual focus | 提供左右视角，用于 RGB 观察和 marker stereo matching。 |
| calibration | GP052 checkerboard 多视角标定 | 获取 intrinsic / extrinsic parameters（内外参）。 |
| tactile interface | Solaris 透明硅胶，1:1 配比，3D 打印模具，厚度 5 mm | 提供可变形且可透视的接触界面。 |
| markers | fluorescent markers 手工嵌入不规则空间位置，薄硅胶封装 | 在 UV 下被检测，作为变形和深度锚点。 |
| illumination | 365 nm UV strip + white LED strip | 分离 marker 观察和 RGB 观察。 |
| housing | matte black 3D printed housing | 抑制内部反射和边界伪影。 |

这一结构和 [[@azulay2024allsight]]、[[@do2022densetact]] 的圆形/曲面大接触面路线不同：TransTac 不强调仿生指尖外形，而强调同一透明界面同时承担视觉和触觉。

### C. Marker Detection and Tracking

传统 blob-based methods 对这种场景不够稳，因为 markers 半透明、密集、受 UV/RGB 切换影响，并且接触形变会改变形状和局部亮度。本文采用 lightweight single-stage, anchor-free detector（轻量单阶段无锚点检测器），基于 compact convolutional backbone，并加入 correlation modeling module（相关性建模模块）捕捉密集 markers 的局部和高阶空间关系。

检测头是 CenterNet-style design：一个 Gaussian-encoded center heatmap branch（高斯编码中心热图分支）负责定位 marker center，另一个 regression branch（回归分支）精修 bounding box。skip connections 用来保留小 marker 的 fine-grained spatial details（细粒度空间细节）。

训练损失为：

$$
\mathcal{L} =
w_{\text{loc}} \mathcal{L}_{\text{loc}}
+ w_{\text{reg}} \mathcal{L}_{\text{reg}}
+ w_{\text{corr}} \mathcal{L}_{\text{corr}}
$$

其中：

- $\mathcal{L}_{\text{loc}}$ 监督 marker center 的高斯热图定位。
- $\mathcal{L}_{\text{reg}}$ 精修 bounding box geometry。
- $\mathcal{L}_{\text{corr}}$ 约束同一 local cluster（局部簇）内 marker feature consistency（特征一致性）。
- $w_{\text{loc}}, w_{\text{reg}}, w_{\text{corr}}$ 是三项损失权重。

检测完成后，ByteTrack 用于 across-frame association（跨帧关联），保持 marker identity（身份一致性），并用 displacement（位移）估计触觉变形。

### D. Prior-Guided Stereo Marker Matching

匹配问题形式化如下：给定 rectified stereo images $\mathbf{I}^L$ 和 $\mathbf{I}^R$，左右图 marker centers 分别是

$$
\mathcal{P}^L = \{\mathbf{p}^L_i\}, \quad
\mathcal{P}^R = \{\mathbf{p}^R_j\}
$$

目标是建立可靠对应：

$$
\mathcal{M} = \{(\mathbf{p}^L_i,\mathbf{p}^R_j)\}
$$

#### 1. Epipolar Prior Initialization

先用极线约束筛选右图候选点：

$$
|y^L_i - y^R_j| \le \tau_{\mathrm{epi}}
$$

也就是说，左图某个 marker 的右图候选应该在近似同一行，误差不超过 $\tau_{\mathrm{epi}}$。在候选中，算法选择 disparity space（视差空间）最近邻，形成初始 correspondence set（对应集合）$\mathcal{M}_0$。这些对应不一定全部正确，但作为 high-confidence priors（高置信先验）帮助后续找 anchor。

#### 2. Delaunay Encoding and Anchor Discovery

算法对左右点集分别做 Delaunay triangulation，得到三角集合 $\mathcal{T}^L$ 和 $\mathcal{T}^R$。对三角形 $t_{ijk}$，定义归一化边长描述子：

$$
\mathbf{d}_{ijk} =
\frac{
[\|\mathbf{p}_i-\mathbf{p}_j\|,\;
 \|\mathbf{p}_j-\mathbf{p}_k\|,\;
 \|\mathbf{p}_k-\mathbf{p}_i\|]
}{
\|[\|\mathbf{p}_i-\mathbf{p}_j\|,
  \|\mathbf{p}_j-\mathbf{p}_k\|,
  \|\mathbf{p}_k-\mathbf{p}_i\|]\|_2
}
$$

这个描述子不直接依赖绝对位置，而依赖局部三角形的相对边长结构。满足 descriptor similarity（描述子相似）和 epipolar consistency（极线一致性）的 triangle pairs 被选为 anchor seeds（锚点种子）。

#### 3. Triangle Propagation

从 anchor pairs 出发，算法向相邻三角形传播对应关系，并持续检查 geometric consistency（几何一致性）。直观上，它不把每个 marker 当成孤立小点，而是利用“邻居拓扑大致保持”这一弹性体形变先验。这样可以减少重复 marker 外观导致的 global assignment ambiguity（全局分配歧义）。

### E. RGB-D Fusion with Scale Optimization

本文并不是只重建稀疏 marker 点。作者还使用 FoundationStereo 预测 dense depth map（稠密深度图），但这类网络在近距离可能存在 scale bias（尺度偏差）。TransTac 使用 marker triangulation 提供 metric sparse anchors（公制稀疏锚点），对 dense map 做尺度校正。

距离切换阈值 $\tau$ 定义为 stereo cameras 和 elastomer surface 之间的标定距离：

- 当 $d > \tau$，视觉 stereo features 提供 sparse anchors。
- 当 $d < \tau$，使用可检测位移的 triangulated markers 作为 anchors。

给定匹配 marker pairs $(p_i^L,p_i^R)$ 和 stereo projection matrices：

$$
P_L = K_L[I|0],\quad P_R = K_R[R|t]
$$

每个对应点被三角化为：

$$
X_i^{\text{sparse}}=(X_i,Y_i,Z_i)
$$

这里 $K_L$ 是左相机内参，$R,t$ 描述右相机相对左相机的旋转和平移，$Z_i$ 是 marker 的深度。

同时，dense depth model 预测每个像素的深度 $D_{net}(u,v)$，并反投影成三维点：

$$
X_{dense}(u,v)=
\left(
\frac{u-c_x}{f_x}Z,
\frac{v-c_y}{f_y}Z,
Z
\right),\quad Z=D_{net}(u,v)
$$

其中 $(u,v)$ 是像素坐标，$(c_x,c_y)$ 是主点，$(f_x,f_y)$ 是焦距。对每个 triangulated feature location，取对应 dense point，形成 sparse-dense correspondences。

经过 RANSAC 去除 outliers 后，估计 similarity transformation：

$$
T=(s,R,t)
$$

并通过 Umeyama alignment 最小化：

$$
\sum_i \|X_i^{sparse}-(sRX_i^{dense}+t)\|^2
$$

这个公式的直观含义是：寻找一个尺度 $s$、旋转 $R$ 和平移 $t$，让网络给出的 dense points 尽量贴合 marker 三角化得到的公制 sparse points。最后把这个变换应用到整张 dense map 上，得到 metrically consistent dense depth map（公制一致的稠密深度图）。

## IV. Experiments and Results / 实验设置与结果

实验验证四类能力：

- semantic recognizability of tactile imagery（触觉图像语义可识别性）。
- recovery of near-contact geometry（RGB-D 失效区域的近接触几何恢复）。
- robustness of stereo marker correspondence estimation（双目 marker 对应鲁棒性）。
- stability of tactile marker tracking（接触/滑动过程中的 marker 跟踪稳定性）。

几何重建实验使用 3D-printed objects 的 STL models 作为 ground-truth geometry，避免依赖噪声较大的主动深度测量。定量实验中，传感器固定在 three-axis gantry system（三轴龙门平台）上，Z 方向重复定位达到 sub-millimeter precision（亚毫米精度）。定性实验使用手持传感器，覆盖不同接触姿态、滑动和真实操作变化。

### A. Semantic Recognizability of Tactile Images

这组实验问的是：TransTac 的 tactile images 是否仍像 natural images 一样保留物体类别线索？比较对象是 GelSight Mini、9DTact 和 TransTac。对象类别包括 egg、coin、battery、Lego block、button、glass bead。每类两个实例，每个实例三个视角，所以每种 modality 有 36 张 tactile images。

评测模型分两类：

- vision-language models：Qwen-VLM、ChatGPT-VLM。
- open-vocabulary detectors：YOLO-World、YOLO-E。

因为触觉图像可能只包含局部物体线索，作者使用 weighted semantic scoring scheme（加权语义评分）：部分匹配 ground-truth category 的预测也给比例分，而不是全错。

**Table I: Semantic preservability across tactile modalities.**

| Model | GelSight | 9DTact | TransTac |
| --- | ---: | ---: | ---: |
| Qwen-VLM | 28.7 | 10.8 | 80.6 |
| ChatGPT-VLM | 30.2 | 12.5 | 83.3 |
| YOLO-World | 15.3 | 2.1 | 72.2 |
| YOLO-E | 16.9 | 3.6 | 75.0 |
| SigLIP2 (Zero-shot %) | 27.8 | 27.8 | 83.3 |
| DINOv2 (Center Similarity) | 0.236 | 0.202 | 0.774 |
| DINOv2 (NN Top-1 %) | 27.8 | 55.6 | 100.0 |

这个表是全文最能体现“transparent tactile sensor（透明触觉传感器）”价值的证据。GelSight 和 9DTact 的图像对几何重建有用，但对视觉语义模型来说远离自然图像分布；TransTac 因为透明，接触界面仍保留颜色、轮廓和材质线索，所以跨模态表征更接近自然图片。

需要注意样本规模：36 images per modality 并不大，所以这个实验更像 proof-of-concept（概念验证），说明透明触觉图像有强语义潜力，但还不是大规模识别 benchmark。

### B. Near-Contact Geometry Recovery

![[papers/images/2606.04477v1/result_2_a.png|700]]

**Fig. 3(a): Scale ambiguity in learning-based methods.** 作者比较 VGGT 和 FoundationStereo 等 depth estimation models。VGGT 容易把 perspective cues（透视线索）当作平面投影，不能恢复正确 metric scale；FoundationStereo 保留相对结构，但由于 stereo baseline（双目基线）短和 distribution mismatch（分布不匹配），仍有绝对尺度误差。

![[papers/images/2606.04477v1/result_2_b.jpg|700]]

**Fig. 3(b): Limitations of coated VBTS in perceiving recessed geometry.** 传统 coated VBTS 的问题不是“看不清接触凸起”，而是无法看见没有真正接触到传感器的 recessed geometry。只有凸出区域压到弹性层时才有触觉信号；凹陷结构、边界锐角会因为弹性体形变被平滑成斜坡，产生 false depths（伪深度）。

![[papers/images/2606.04477v1/depth_coverage_alignment.png|700]]

**Fig. 4: Near-contact depth sensing and alignment evaluation.** Fig. 4(a) 测量 Intel RealSense D405 与平面物体距离变化时 valid depth ratio（有效深度比例）。当距离大于约 9 cm 时覆盖较稳定；进入 near-contact region 后有效深度急剧下降，最后低于 10%。Fig. 4(b) 显示经过 RANSAC filtering 和 sparse-dense alignment 后，marker triangulation 与 dense depth 的 mean geometric error 约 2.44 mm。

这组实验支撑的是本文最核心的“过渡”叙事：RGB-D 在接触前最后几厘米不可靠，但 marker triangulation 仍能提供几何锚点；coated VBTS 到接触后才有效，但无法看非接触凹陷。TransTac 则把透明视觉和 marker 触觉合并，试图覆盖从 approach 到 contact 的连续区域。

### C. Stereo Marker Matching Robustness

双目 marker matching 是几何重建的关键。如果左右图 marker 对应错了，三角化深度就会错。本文比较四类基线：

- Epipolar Nearest Neighbor：只用极线约束和最近邻。
- Hungarian Assignment：全局分配。
- Optical Flow (Farneback)：稠密光流。
- Stereo Matching (SGBM)：传统稠密双目匹配。

benchmark 由不同接触压力、滑动、局部遮挡下的 stereo marker observations 构成，ground truth correspondences 由人工标注。

**Table II: Stereo marker correspondence comparison.**

| Method | Avg. Correct Matches |
| --- | ---: |
| Prior-Guided Delaunay (Ours) | 90.8 |
| Hungarian Assignment | 74.9 |
| Epipolar Nearest Neighbor | 74.5 |
| Optical Flow (Farneback) | 37.9 |
| Stereo Matching (SGBM) | 28.7 |

Prior-Guided Delaunay 比 Hungarian 多 15.9 个正确匹配，约提升 21.2%；比 epipolar nearest neighbor 多 16.3 个正确匹配。dense correspondence methods 表现很差，原因是 near-contact stereo geometry 下左右相机看到的可能不是同一物理表面，某些区域在一侧可见、另一侧不可见，导致密集匹配假设被破坏。

### D. Tactile Marker Tracking

![[papers/images/2606.04477v1/result_3.jpg|700]]

**Fig. 5: Qualitative comparison.** 左侧比较 marker detection：传统 color thresholding + blob detection 在照明变化、局部遮挡和接触形变下容易 missed detections（漏检）或不稳定；本文 detector 更稳定。右侧比较 trajectory stability：optical flow 容易 cumulative drift（累积漂移）、identity swaps（身份交换）和 intermittent loss（间歇丢失），而本文通过 detection + ByteTrack 维持 marker identity。

作者报告 detection model 在 NVIDIA RTX 4060 laptop GPU 上可 20 FPS 实时运行。这个数字对硬件入库很重要：它说明 TransTac 当前不是 120 Hz 高频触觉路线，而更偏向 multimodal near-contact perception（多模态近接触感知）和语义/几何连续性。若和 [[@li2025r-tac0]] 比较，R-Tac0 强在 120 Hz 单通道曲面重建，TransTac 强在透明化、视觉语义保留和近接触融合。

## 6. 结果、对比与消融

本文没有传统意义上逐模块 ablation table（消融表），但有几个对比承担了类似功能：

- 与 opaque tactile modalities 的语义对比：证明透明界面保留物体语义，不只是触觉几何。
- 与 RGB-D / learning-based depth 的近距离对比：证明单靠视觉深度在接触前后不稳定。
- 与 coated VBTS 的定性重建对比：证明传统接触界面无法恢复 recessed geometry。
- 与 Hungarian / epipolar NN / optical flow / SGBM 的匹配对比：证明 Delaunay local topology prior 是匹配鲁棒性的关键。
- 与 threshold/blob detection 和 optical flow tracking 的定性对比：证明 learned detector + tracker 比传统视觉方法更适合半透明 marker 形变场。

如果把 Delaunay matching 当作核心算法，其“消融”近似为 Table II 的 Hungarian 和 epipolar NN 两个 baseline：epipolar NN 只用极线先验，Hungarian 用全局分配，而本文加入局部三角结构和传播。因此，性能提升说明局部拓扑先验确实解决了 dense repetitive markers 的歧义。

## 7. 讨论、局限与可追问点

作者在 Limitation and Future Work 中列了四类限制。

第一，TransTac 目前主要是 geometric sensing，不直接测 force or pressure distributions。它能跟踪 marker displacement 和恢复接触表面，但没有像 [[@azulay2024allsight]] 或 [[@lin20239dtact]] 那样直接把输出扩展到 3D force / torsion / 6D force estimation。未来可能需要 elastomer deformation modeling（弹性体形变建模）或 embedded sensing elements（嵌入式传感元件）。

第二，marker detector 需要 annotated training data。对每个新 marker pattern、材料批次、照明条件，都可能需要新的标注或 domain adaptation。文中没有给出标注规模、检测 precision / recall，这会影响复现判断。

第三，dense depth 依赖现有模型，而现有 depth estimation models 通常缺少 extreme close-range / manipulation scenarios 的训练数据。因此即使有 marker anchors，dense map 的细节仍可能受网络先验影响。这个问题在 near-contact robotics 中很关键。

第四，硬件还不是最终机械手指形态。off-the-shelf stereo cameras 和灯带让原型容易搭建，但体积、延迟、曝光切换和布线都还需要进一步工程化。UV fluorescent markers 的手工/半自动制作也可能影响 manufacturability（可制造性）和 long-term durability（长期耐久性）。

可追问点：

- 标记检测数据集有多大？是否跨传感器、跨照明、跨材料批次泛化？
- Delaunay matching 在大形变或 marker 局部丢失时的失败边界在哪里？
- 2.44 mm sparse-dense alignment error 对具体抓取任务是否足够？
- semantic recognizability 的 36 images per modality 是否能扩展到更多材料、遮挡和复杂物体？
- time-multiplexed illumination 的延迟会不会影响快速滑移或动态接触？
- 透明弹性体和 UV markers 在长期摩擦、污染、老化后是否保持可检测性？

## 贡献与结论对照

结论重申 TransTac 不是替代 conventional vision or tactile systems，而是 complementary sensing configuration（互补感知配置）：它保留 visual transparency，同时支持 metric contact reconstruction 和 appearance consistency。

对应全文证据：

- 硬件结构支持同一设备内的视觉观察和触觉 marker 重建，见 Fig. 1-Fig. 2。
- 语义实验说明透明触觉图像与自然图像表征更接近，见 Table I。
- 近距离实验说明 RGB-D 在接触前区域失效，marker triangulation 可作为几何锚点，见 Fig. 4。
- 匹配实验说明 Delaunay 结构先验比全局分配和极线最近邻更鲁棒，见 Table II。
- 跟踪实验说明 learned detection + ByteTrack 比 threshold/blob 和 optical flow 更稳定，见 Fig. 5。

总体判断：TransTac 的贡献不在最高精度触觉重建，而在提出一个“视觉可穿透 + UV marker 可重建 + 双目近接触融合”的传感器方向。它适合与 GelSight/DTact/R-Tac0 对读：这些工作把接触界面做成稳定的触觉成像面；TransTac 则牺牲一部分传统触觉封闭性，换取外部视觉语义和近接触连续几何。
