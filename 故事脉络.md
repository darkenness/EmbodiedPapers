# 初步论述逻辑
- [[@lin2022dtact|DTact]] 证明了单通道强度映射可以避免 RGB 光度立体的复杂性，用较低成本实现触觉表面 3D 重建。
- [[@lin2022dtact|DTact]] / [[@lin20239dtact|9DTact]] 的核心优势在于低成本、单通道、查表式重建；但这种全局 LUT 隐含了一个近似：同一灰度差在不同空间位置对应相近的深度响应。
- [[@lin2022dtact|DTact]] 中完整映射本身可以写成与位置相关的形式，但为了降低标定成本，最终采用了全局 LUT 或较低维的空间近似。  
- 我们的目标不是放弃 [[@lin2022dtact|DTact]] 的低成本路线，而是在保留单通道 LUT 简洁性的同时，补偿真实器件中不可忽略的空间非均匀性。
- 我们采用外反射层光学结构，是为了让接触形变更直接地调制回光强度，使传感器形成“越深越亮”的单通道响应。  
- 但这种响应不是只由压入深度决定，还会受到光带位置、入射角、相机视角、反射层涂布、局部材料状态和弹性体形变的共同影响；因此更准确地说，空间非均匀性来自整个外反射光学栈，而不是单独来自反射层。
- **问题变成了：需要解决在外反射层深亮型单通道传感器中，如何用低标定成本补偿空间相关的亮度-深度响应差异。**
- [[@wang2021gelsight-wedge|GelSight Wedge]]、[[@wang2024large-scale-vbts|Large-scale Deployment]]、[[@kota2026-3dcal|3D Cal]] 说明，视觉触觉传感器中的强度/颜色映射往往不是纯像素值函数，而会受到空间位置、光照传播和装配差异影响。
- 但这些工作多面向 RGB 光度立体、RGB/RGB-diff 到梯度映射，或者使用 MLP/CNN 进行连续映射；我们的目标是在单通道 LUT 框架内，用更轻量、可解释、可部署的形式引入空间校正。
- 我们提出 SA-LUT（Spatially Adaptive LUT）：在全局单调 LUT \(g(diff)\) 的基础上，引入低维空间校正场 \(a(x,y), b(x,y)\)：  
$$  
\hat d = a(x,y)g(diff)+b(x,y)  
$$
- 这样做相当于把完整空间相关映射 \(f(diff,x,y)\) 约束成“全局单调响应 + 平滑空间增益/偏置”的形式，避免高成本逐像素 LUT，也避免直接使用大网络。  
- [[@li2025r-tac0|R-Tac0]] 说明单通道/黑白视觉触觉传感器在高频、低计算和可迁移标定上有现实价值；这进一步支持单通道 + 低成本 + 可部署的方向。  
- 和 [[@li2025r-tac0|R-Tac0]] 这类轻量网络方法相比，SA-LUT 更强调可解释性和部署简洁性：推理时只需要查表、空间插值和一次乘加。  
- (**做不做需要进一步确定**)为了提高球压标定中接触圆边界的可靠性，可以进一步借鉴 [[@redkin2024dynamic-illumination|Dynamic Illumination]] 和 [[@raskar2004non-photorealistic-camera|Multi-Flash]] 的思想，用同一光带不同光强差分增强圆边缘，降低圆心/半径检测误差。  
- 因此，当前故事可以概括为：外反射光学栈提供单通道深亮编码，SA-LUT 补偿空间非均匀性，高低光强差分提高标定边缘可靠性，最终目标是在不显著增加标定和部署成本的情况下提升深度重建精度。
# 核心gap
  
- [[@lin2022dtact|DTact]] / [[@lin20239dtact|9DTact]] 证明了单通道 LUT 的低成本优势，但主要依赖全局灰度-深度映射，空间非均匀性处理较弱。  
- [[@wang2021gelsight-wedge|GelSight Wedge]] / [[@kota2026-3dcal|3D Cal]] 证明显式空间位置对视觉触觉映射有帮助，但主要面向 RGB 光度立体或学习式梯度/深度恢复。  
- [[@li2025r-tac0|R-Tac0]] 证明单通道视觉触觉可以结合轻量网络实现高频和可迁移重建，但其路线更偏向学习式模型。  
- 当前缺口是：面向外反射层深亮型单通道传感器，如何在不引入高成本逐像素 LUT 或大网络的情况下，显式补偿空间非均匀性。  
- SA-LUT 的定位就是填补这个中间空缺：比 Global LUT 更能处理空间差异，比逐像素 LUT 更低标定成本，比大网络更可解释、更容易部署。
# 方法部分主线

- Step 1：采集参考图和球压样本，得到原始差分 \(diff_{raw}=I-I_{ref}\)，并通过统一预处理得到 LUT 输入 \(diff\)。
- Step 2：由人工/半自动圆标注得到接触圆心和半径，根据球面几何生成密集深度标签 \(d_{geo}\)。
- Step 3：构建全局单调 LUT \(g(diff)\)，得到基础深度：
  $$
  d_{base}=g(diff)-g(diff_0)
  $$
  其中 \(diff_0\) 是非接触基准。
- Step 4：在 ROI 内拟合低维空间校正场：
  $$
  d_{geo} \approx a(x,y)d_{base}+b(x,y)
  $$
- Step 5：实时推理时，先通过全局 LUT 得到 \(d_{base}\)，再通过 \(a(x,y),b(x,y)\) 做空间修正：
  $$
  \hat d=a(x,y)d_{base}+b(x,y)
  $$
- Step 6：用 Global / Radial / Tile SA-LUT 做消融，验证空间建模的必要性。
# 实验部分回答问题
- 实验要回答的问题：
 - **全局 LUT 是否足够？**  
	- 看不同位置的 \(diff \rightarrow depth\) 曲线是否重合；  
	- 看 Global residual heatmap 是否有空间结构。  
- （*？？？-需要验证*）**空间非均匀性是否是结构性问题？**  
	- 看不同样本/不同深度下 residual heatmap 是否呈现稳定模式；  
	- 看 gain grid 是否是平滑场，而不是棋盘格噪声；  
	- 看空间误差是否和光带方向、中心-边缘位置或涂层状态有关。  
- **径向修正是否足够？**  
	- 比较 Global LUT、Radial LUT、Tile SA-LUT；  
	- 如果 Radial 已经接近 Tile，说明主要是中心-边缘差异；  
	- 如果 Tile 明显优于 Radial，说明存在非径向空间不均匀性。  
- **SA-LUT 是否真的提升？**  
	- 比较 Global / Radial / Tile 的 MAE、RMSE、edge MAE；  
	- 使用按 sample_id 拆分的验证集，避免同点拟合、同点评估。  
- **加 \(x,y\) 是否增加太多标定代价？**  
	- 做 \(3\times3 / 5\times5 / 7\times7 / dense\) 标定密度消融；  
	- 看低密度空间采样是否已经接近 dense 标定效果。  
- **高低光强差分是否有用？**  
	- 比较原始 diff 与高低光强差分下的圆检测成功率、圆心误差、半径误差。
