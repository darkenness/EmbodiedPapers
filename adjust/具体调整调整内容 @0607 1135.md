# 1. 当前故事最大的问题

你现在的故事大概是：

```text
DTact 证明单通道 LUT 可行
↓
但全局 LUT 有空间误差
↓
其他工作说明可以加入 x,y
↓
所以我提出 SA-LUT
```

这个逻辑是成立的，但**不够强**，主要有四个问题。

---

## 1.1 问题太像算法补丁

你现在的核心问题容易被理解成：

```text
我的 Global LUT 精度不够，所以我加了 x,y。
```

这听起来像一个 engineering fix，而不是一个论文级问题。

应该改成：

```text
低成本单通道触觉重建存在一个结构性矛盾：
Global LUT 简单、低成本、易部署；
但真实外反射光学栈的亮度—深度响应具有空间非均匀性；
而逐像素 LUT 或大网络又会破坏低标定成本和简便部署。
```

这样问题就变成了一个更高层的 trade-off：

[  
\text{accuracy} \quad vs \quad \text{calibration cost / deployability}  
]

SA-LUT 就不是“加两个变量”，而是解决这个 trade-off 的中间解。

---

## 1.2 外反射层目前像“问题来源”，不像“贡献来源”

你现在容易写成：

```text
我们用了外反射层，所以空间不均匀更严重，所以要修正。
```

这样会让读者觉得：你先制造了一个问题，然后自己解决。

应该改成：

```text
外反射层的作用是提供一种新的单通道深度编码方式：深压更亮。
```

也就是：

```text
外反射层 = 单通道深亮编码的物理基础
空间非均匀性 = 真实光学栈中的副作用
SA-LUT = 与这种光学编码匹配的低成本反映射
```

这样硬件结构才是贡献，而不是包袱。

---

## 1.3 (x,y) 的意义现在讲得太像“别人做过，所以我也做”

你引用 GelSight Wedge、Large-scale VBTS、3D Cal 是对的，但表达要小心。

不要写成：

```text

```

应该写成：

```text
这些工作说明视觉触觉映射中的空间依赖是普遍问题；但它们多用于 RGB/RGB-diff 到梯度/深度的学习式映射。我们的目标是在单通道 LUT 框架内，用低维可部署的 gain/bias 场补偿空间依赖。
```

差别在于：

- 别人：position information 是网络输入；
    
- 你：position information 被压缩成低维、平滑、可查表部署的空间校正场；
    
- 别人：多通道 / RGB / MLP / CNN；
    
- 你：单通道 / LUT / 乘加 / 低标定成本。
    

这才是你的边界。

# 3. 新故事逻辑：按论文 Introduction 的顺序

下面这版可以直接放进 Obsidian，作为“新故事主线”。

```markdown
# 新故事逻辑

- 视觉触觉传感器的一个核心目标是以低成本恢复接触表面的高分辨率几何。
- RGB GelSight 系通常依赖多方向彩色照明和光度立体，精度高但光路、膜层和标定复杂。
- [[@lin2022dtact|DTact]] / [[@lin20239dtact|9DTact]] 证明了另一条路线：通过单通道强度/暗度映射，可以绕开 RGB 光度立体，用较低成本实现触觉深度重建。
- 但 Global LUT 隐含一个关键近似：同一灰度差在不同空间位置对应相近深度。
- 对于我们采用的外反射层结构，压入形变会增强局部回光，使传感器呈现“越深越亮”的单通道响应；这为低成本单通道深度重建提供了物理基础。
- 但真实响应并不只由压入深度决定，而受到外反射层、光带位置/角度、涂层工艺、弹性体形变、相机视角和内部反射共同影响，因此亮度—深度映射具有空间非均匀性。
- 完整建模可以写成：
  $$
  d=f(diff,x,y)
  $$
  但逐像素 LUT 或端到端网络会增加标定数据、部署复杂度和解释成本。
- 因此本文关注的问题是：如何在保留单通道 LUT 低成本和可部署优势的同时，补偿外反射光学栈带来的空间非均匀性。
- 我们提出 SA-LUT（Spatially Adaptive LUT），将完整映射分解为：
  $$
  \hat d=a(x,y)g(diff)+b(x,y)
  $$
- 其中 \(g(diff)\) 是全局单调亮度—深度 LUT，表示外反射层提供的主深亮响应；\(a(x,y),b(x,y)\) 是低维平滑空间校正场，表示位置相关灵敏度和偏置。
- 该形式比 Global LUT 更能补偿空间误差，比逐像素 LUT 标定成本更低，比大网络更可解释、更易部署。
- 为了保证球压标定标签可靠，本文进一步采用人工可复核圆标注，并探索同一光带高低光强差分增强接触圆边缘。
```

本文面向低成本单通道视触觉深度重建，研究一种外反射层传感器结构及其空间自适应标定方法。外反射光学栈使压入区域产生“越深越亮”的单通道响应，从而避免 RGB 光度立体的复杂照明和颜色标定；但由于光带位置、反射层涂布、弹性体形变和相机视角共同影响，同一亮度差在不同空间位置并不对应相同深度，导致全局 LUT 产生结构性空间误差。为在精度和标定/部署成本之间取得平衡，本文提出 SA-LUT，将深度映射分解为全局单调 LUT \(g(diff)\) 和低维空间校正场 \(a(x,y),b(x,y)\)。该方法保留查表式推理的简洁性，同时显式补偿位置相关响应差异。进一步地，本文构建人工可复核的球压标定流程，并引入高低光强差分增强接触圆边缘，以提高深度标签可靠性。实验将通过全局、径向和 Tile SA-LUT 的消融，以及标定密度消融，验证 SA-LUT 在低标定成本条件下提升单通道深度重建精度的有效性。
