## 研究域书架

> [!note] 组织原则
> 先按研究域分层，再在每个研究域内按“综述/坐标系、硬件与数据源、表征与模型、任务与评测、部署与复现”归位。每个条目保持 `问题 -> 方法钩子 -> 回看理由`，方便以后把 VLA、WMA 或其他方向并列加入。

### 视触觉传感器（Vision-Based Tactile Sensors, VBTS）

#### 综述与坐标系

- [[@li2025vbts-classification-review|Classification of Vision-Based Tactile Sensors: A Review]] (2025) :: 问题：VBTS 分类容易按外观或品牌混在一起；方法钩子：用 transduction mechanism（换能机制）重分 IMM / MDM / MFM 与 RLB / TLB；回看：给新增传感器归档，判断一篇工作主要在改材料、光路、marker 还是算法。
- [[@abad2020visuotactile-review|Visuotactile Sensors With Emphasis on GelSight Sensor: A Review]] (2020) :: 问题：GelSight 家族历史和组件取舍分散；方法钩子：从 optical pedobarograph（光学足底压力记录）梳理到反射膜、支撑板、照明、相机、marker 和软件；回看：查 GelSight-style 方案的历史脉络和硬件演化。

#### 接触几何重建：反射膜 / 光度立体

- [[@yuan2017gelsight|GelSight: High-Resolution Robot Tactile Sensors for Estimating Geometry and Force]] (2017) :: 问题：如何从软表面接触同时获得高分辨率几何、力、剪切和滑移；方法钩子：reflective membrane、RGB lighting、photometric stereo 与 markers；回看：作为 GelSight-style VBTS 的基线和术语入口。
- [[@dong2017improved-gelsight|Improved GelSight Tactile Sensor for Measuring Geometry and Slip]] (2017) :: 问题：早期 GelSight 的膜层和照明影响 surface normal 精度与滑移判断；方法钩子：更 Lambertian（朗伯）的膜层、重做三色照明、物体纹理与 marker 相对运动；回看：理解膜层/照明如何影响 RGB-to-normal mapping。
- [[@donlon2018gelslim|GelSlim: A High-Resolution, Compact, Robust, and Calibrated Tactile-sensing Finger]] (2018) :: 问题：GelSight 光路不容易直接塞进耐用的机器人手指；方法钩子：导光、镜面折叠、织物皮肤、在线校准；回看：看真实抓取中的材料老化、光强下降和校正流程。
- [[@wang2021gelsight-wedge|GelSight Wedge: Measuring High-Resolution 3D Contact Geometry with a Compact Robot Finger]] (2021) :: 问题：紧凑 wedge finger 的空间非均匀成像使全局标定困难；方法钩子：RGBXY-to-gradient MLP 加 Poisson solver；回看：引用 pixel position（像素位置）作为显式输入处理非均匀光路。

#### 接触几何重建：灰度 / 暗度 / 阴影

- [[@lin2022dtact|DTact: A Vision-Based Tactile Sensor that Measures High-Resolution 3D Geometry Directly from Darkness]] (2023) :: 问题：RGB 光度立体成本和标定复杂度高；方法钩子：半透明层受压变暗的 darkness mapping，把灰度差查表成 depth map；回看：作为 low-cost global LUT / single-image calibration 的起点。
- [[@lin20239dtact|9DTact: A Compact Vision-Based Tactile Sensor for Accurate 3D Shape Reconstruction and Generalizable 6D Force Estimation]] (2023) :: 问题：低成本深度重建如何扩展到小型硬件和 6D 力估计；方法钩子：DTact 灰度差深度重建、dense deformation representation、6D force regression；回看：看几何表征如何服务力估计泛化。
- [[@lu2024tacshade|TacShade: A New 3D-printed Soft Optical Tactile Sensor Based on Light, Shadow and Greyscale for Shape Reconstruction]] (2024) :: 问题：能否用更简单的黑白图案和阴影线索做触觉形状重建；方法钩子：TacTip 结构、light / shadow / greyscale 变化、粗 2.5D/3D reconstruction；回看：借用灰度边缘和 shape-from-shading 思路。
- [[@li2025r-tac0|R-Tac0: A Rounded High-Frequency Transferable Monochrome Vision-based Tactile Sensor for Shape Reconstruction]] (2025) :: 问题：单通道、曲面和高频触觉重建能否兼顾迁移；方法钩子：黑白相机、白光、复合圆弧弹性体、120 Hz reconstruction；回看：比较快速部署、跨传感器迁移和曲面形态的取舍。

#### 曲面形态与接触状态估计

- [[@do2022densetact|DenseTact: Optical Tactile Sensor for Dense Shape Reconstruction]] (2022) :: 问题：平面传感器难覆盖大接触角度和曲面接触；方法钩子：半球形 soft elastomer、鱼眼相机、监督学习输出 radial depth image 和点云；回看：看曲面传感器如何收集 3D 打印压头真值。
- [[@do2023densetact-2|DenseTact 2.0: Optical Tactile Sensor for Shape and Force Reconstruction]] (2023) :: 问题：同一内部图像能否同时支撑形状和力/力矩估计；方法钩子：缩小 DenseTact 1.0、随机纹理、6-axis wrench estimation；回看：比较 shape reconstruction 与 net force / torque 的联合建模。
- [[@azulay2024allsight|AllSight: A Low-Cost and High-Resolution Round Tactile Sensor with Zero-Shot Learning Capability]] (2024) :: 问题：圆形大接触面能否低成本估计完整接触状态；方法钩子：3D 打印圆形光学触觉传感器、3D contact position、3D force、torsion、zero-shot / few-shot transfer；回看：看圆形形态与跨传感器学习。

#### 透明触觉与近接触过渡

- [[@yang2026transtac|TransTac: Visuo-Tactile Modality Transition via Ultraviolet-Encoded Transparent Elastomers]] (2026) :: 问题：传统 coated VBTS 看不到接触前视觉语义，也容易丢失凹陷几何；方法钩子：透明硅胶、UV-reflective markers、双目相机、prior-guided Delaunay stereo matching；回看：理解 vision-through tactile sensing（可透视触觉）和 near-contact perception（近接触感知）的连续表征。

#### 标定、制造、设计与部署

- [[@kota2026-3dcal|3D Cal: An Open-Source Software Library for Calibrating Tactile Sensors]] (2026) :: 问题：逐个手工标定 DIGIT / GelSight Mini 成本高；方法钩子：低成本 FDM 3D printer 改自动 probing device，TouchNet 从 RGB + xy 预测 gradients；回看：查自动标定、坐标通道和少量采样消融。
- [[@fan2025crystaltac|CrystalTac: Vision-Based Tactile Sensor Family Fabricated via Rapid Monolithic Manufacturing]] (2025) :: 问题：不同 VBTS 机制难以用一致工艺低成本复现；方法钩子：多材料 3D printing 一体化制造传感器 family；回看：看 fabrication consistency、批量成本和传感器族设计。
- [[@agarwal2025pbr-design|Vision-based tactile sensor design using physically based rendering]] (2025) :: 问题：触觉传感器光学结构设计依赖 trial-and-error；方法钩子：physically based rendering 模拟光源、几何、厚度和涂层材料，用 RGB2Normal objective 评价设计；回看：把硬件设计问题转成仿真优化问题。
- [[@wang2024large-scale-vbts|Large-scale Deployment of Vision-based Tactile Sensors on Multi-fingered Grippers]] (2024) :: 问题：多指夹爪上部署多个 GelSight-style 模块会遇到同步和逐个标定成本；方法钩子：7 个模块、三指 GelGripper、hub board 同步采集、zero-shot / diff input calibration；回看：看多传感器系统部署、同步和迁移标定。

### 视觉成像与可迁移方法

#### 多照明、边缘线索与图像融合

- [[@raskar2004non-photorealistic-camera|Non-Photorealistic Camera: Depth Edge Detection and Stylized Rendering Using Multi-Flash Imaging]] (2004) :: 问题：普通图像很难直接稳定提取 depth discontinuity；方法钩子：多个 flash 制造 shadow sliver，用 ratio image 检测 depth edges；回看：借用“多照明直接产生几何边界 primitive”的思想。
- [[@redkin2024dynamic-illumination|Enhance Vision-based Tactile Sensors via Dynamic Illumination and Image Fusion]] (2024) :: 问题：固定照明下 DIGIT 图像 contrast、sharpness 和 background difference 受限；方法钩子：切换 RGB LED patterns 并融合多帧；回看：评估动态照明带来的质量提升与延迟代价。

### 多模态模型：高效推理与视觉 Token 压缩

#### Elastic visual-token compression

- [[@kuzucu2026parcel|PARCEL: Pool-Anchored Resampling with Conditioned Elastic Queries for Efficient Vision-Language Understanding]] (2026) :: 问题：LVLM 的 visual tokens 让 decoder prefill、attention 和 KV-cache 成本随图像/视频长度快速上升，而单一路径压缩要么 spatial pooling 丢高频细节、要么 query resampling 丢空间 grounding；方法钩子：Pool-Anchored Resampling + Pool-Conditioned Query Resampling（PCQR），用 4x4/8x8 spatial pool anchors 承担低频布局，再用 conditioned elastic query tokens 补充高频/语义细节；回看：研究 train-once-deploy-anywhere 视觉 token 预算、长视频/高分辨率文档推理、VLA/VLM 高效视觉前端和空间 grounding 与 token compression 取舍时的入口。

### 具身智能：VLA（Vision-Language-Action）

#### 具身基础模型与 VLA 骨干

- [[@tencent2026hy-embodied-05|HY-Embodied-0.5: Embodied Foundation Models for Real-World Agents]] (2026) :: 问题：通用 VLM 缺少机器人所需的细粒度视觉、3D/时空感知和具身推理，直接接入 VLA action expert 时不一定能支撑真实操作；方法钩子：HY-ViT 2.0、Mixture-of-Transformers、visual latent tokens、具身/空间/感知大规模数据、GRPO/RFT 后训练和 large-to-small on-policy distillation，再把 MoT-2B 接 Action Expert 做 HY-Embodied-0.5 VLA；回看：研究 embodied VLM backbone、VLA 前置认知模型、UMI 预训练到真实机器人 SFT 的迁移链条时的入口。
- [[@lin2026physbrain|PhysBrain: Human Egocentric Data as a Bridge from Vision Language Models to Physical Intelligence]] (2026) :: 问题：VLA 需要第一视角手-物-任务关系和动作时序，但大规模机器人数据昂贵，通用 VLM 又常受 third-person / web data 视角偏差限制；方法钩子：Egocentric2Embodiment pipeline 从 Ego4D、BuildAI/Egocentric-10K、EgoDex 生成约 3M 条 E2E-3M 结构化 VQA，用 rule checker 约束 evidence grounding、egocentric consistency 和 temporal logic，再 SFT 得到 PhysBrain，并作为 PhysVLA 的 System-2 backbone 条件化 Flow-Matching DiT action expert；回看：研究 human egocentric video -> embodied VLM brain、VLA few-shot / sample-efficient fine-tuning、第一视角物理过程理解和机器人数据替代来源时的核心入口。

#### 数据采集、UMI 与 robot-free demonstrations

- [[@xu2026egoguide|EgoGuide: Egocentric Guidance for Efficient Robot-Free Demonstration Collection and Learning]] (2026) :: 问题：UMI-style robot-free demonstrations 成本低但容易录到冗余成功轨迹，且 wrist-only observation 在遮挡、物体消失和长时程进度判断时缺少全局上下文；方法钩子：EgoGuide-UMI 同步 wrist camera / wrist pose / gripper state / head-egocentric image / head pose，并用 DINOv2+CLIP visual novelty 与 wrist-pose geometric novelty 生成 AR 中的 0-100 coverage guidance，再用 partial demonstration 和 static filtering 改善数据覆盖；GERP 则把 egocentric view 作为 gated residual action cue 融合到稳定 wrist-view base policy；回看：研究 UMI 数据质量、无机器人示范采集、crowdsourced robot data、multi-view policy fusion 和遮挡下 wrist/ego 视角互补时的核心入口。

#### 策略引导、快速适应与策略改进

- [[@tang2026frs|Improving Robotic Generalist Policies via Flow Reversal Steering]] (2026) :: 问题：通用 VLA 已经含有很多合理动作先验，但新任务中语言命令和随机采样不一定能调用正确 action mode；方法钩子：Flow Reversal Steering（FRS）把人或 VLM 给出的粗方向动作沿 flow policy 反向映射成 noise，再正向去噪成分布内精细动作，并用这些 noise 做 DSBC 或 warm-start DSRL；回看：研究 policy steering、noise-space learning、VLM semantic guidance 如何接入机器人通用策略时的核心入口。

#### 状态表征、时空记忆与长时程移动操作

- [[@kim2026serf|SERF: Spatiotemporal Environment and Robot Feature Map for Long-Horizon Mobile Manipulation]] (2026) :: 问题：长时程移动操作需要同时记住机器人位置、对象变化和任务进度，image-only VLA 难以从当前视野恢复这些状态；方法钩子：Spatiotemporal Environment and Robot Feature Map（SERF）把环境和机器人本体都表示为可更新 neural points，用 object-level $\mathrm{SE}(3)$ tracking 和 forward kinematics 更新坐标，再抽多参考系、多尺度 map tokens 接入 $\pi_{0.5}$ VLA；回看：研究 VLA explicit spatial memory、robot-body state tokens、BEHAVIOR-1K 长时程评测和 map-conditioned policy interface 时的核心入口。

#### 评测基准、过程理解与数据集

- [[@processbench2026roboprocessbench|RoboProcessBench]] (2026) :: 问题：机器人 VLM 往往能识别静态物体，却未必理解操作执行过程中的阶段、接触、运动、双臂协调、时间顺序和 primitive 转换；方法钩子：RoboProcessBench 从 GM-100、RH20T、REASSEMBLE、AIST-Bimanual 派生 57,892 条 QA rows，分成 12 个 diagnostic task families，并用 episode/recording isolation 构造 SFT/eval split；回看：评估 VLM/VLA 的 process understanding、temporal reasoning、contact reasoning 和 primitive-level action understanding 时的基准入口。

### 世界模型与行动模型：WMA（World Model / Action Model）

#### 触觉-视觉世界模型与接触丰富规划

- [[@zhang2026contactworld|ContactWorld: What Matters in Vision-Tactile World Models for Contact-Rich Manipulation]] (2026) :: 问题：contact-rich manipulation 的 world model 需要同时处理遮挡、几何约束、接触/摩擦动态和长时程 rollout error，单看 image-only VLA 或 policy success 很难判断到底哪些表征有效；方法钩子：12 个 insertion / disassembly / screwing / exploration 任务、Wrist / Front / PointCloud 三类视觉模态、TacRGB / TacDepth / TacFF 三类触觉模态、JEPA-style latent world model、MPC-CEM goal-conditioned planning，以及 fusion / regularization / encoder / predictor 消融；回看：研究 vision-tactile world model、PointCloud+TacFF 兼容性、接触丰富 latent planning、触觉表征进入 action model 和仿真到真实 tactile representation gap 时的核心入口。
