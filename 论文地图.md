## 按问题线索分类

### 综述与坐标系

这组文献用来判断一篇工作属于哪条 VBTS 传统、在硬件和图像形成机制上解决什么问题。

- [[@li2025vbts-classification-review|Classification of Vision-Based Tactile Sensors: A Review]] (2025): 用 transduction mechanism（换能机制）重分 VBTS，核心是 IMM / MDM / MFM 与 RLB / TLB 等类型；适合给新传感器归档，也适合判断一篇论文是在改材料、光路、marker 还是算法。
- [[@abad2020visuotactile-review|Visuotactile Sensors With Emphasis on GelSight Sensor: A Review]] (2020): 从 optical pedobarograph（光学足底压力记录）梳理到 GelSight，重点是 GelSight 的反射膜、支撑板、照明、相机、marker 和软件应用演化；适合查 GelSight 家族的历史脉络和组件取舍。

### GelSight / 反射膜 / 光度立体主线

这条线把接触几何转成带方向照明的颜色变化，再从 surface normal（表面法线）或 gradients（梯度）积分出 depth / height map。

- [[@yuan2017gelsight|GelSight: High-Resolution Robot Tactile Sensors for Estimating Geometry and Force]] (2017): GelSight 代表性总述；用 reflective membrane、RGB lighting、photometric stereo 和 markers 同时服务 high-resolution geometry、force、shear 和 slip，是理解后续 GelSight-style VBTS 的基线。
- [[@dong2017improved-gelsight|Improved GelSight Tactile Sensor for Measuring Geometry and Slip]] (2017): 把膜层改成更 Lambertian（朗伯）并重做三色照明，提高 surface normal 测量；同时用物体纹理和 markers 的相对运动检测滑移，适合看膜层/照明如何影响 RGB-to-normal mapping。
- [[@donlon2018gelslim|GelSlim: A High-Resolution, Compact, Robust, and Calibrated Tactile-sensing Finger]] (2018): 把 GelSight 光路折进细长 robot finger，强调导光、镜面折叠、织物皮肤、耐久性和在线校准；适合看真实抓取中的材料老化、光强下降和校正流程。
- [[@wang2021gelsight-wedge|GelSight Wedge: Measuring High-Resolution 3D Contact Geometry with a Compact Robot Finger]] (2021): 楔形紧凑手指，用 RGBXY-to-gradient MLP 加 Poisson solver 做 3D contact geometry；关键是把 pixel position（像素位置）显式输入模型，适合作为空间非均匀建模依据。
- [[@wang2024large-scale-vbts|Large-scale Deployment of Vision-based Tactile Sensors on Multi-fingered Grippers]] (2024): 把 7 个 GelSight-style 模块同步装到三指 GelGripper，用 hub board 同步采集，并用 zero-shot / diff input calibration 降低逐个标定成本；适合看多传感器部署、同步和迁移标定。

### Darkness / 灰度 / 低成本深度重建

这条线弱化 RGB 光度立体，把接触深度更多地编码成 brightness / darkness / shadow / greyscale（亮度、暗度、阴影、灰度）变化。

- [[@lin2022dtact|DTact: A Vision-Based Tactile Sensor that Measures High-Resolution 3D Geometry Directly from Darkness]] (2023): 用半透明层受压变暗的 darkness mapping，把灰度差直接查表成 depth map；适合作为 low-cost global LUT / single-image calibration 的故事起点。
- [[@lin20239dtact|9DTact: A Compact Vision-Based Tactile Sensor for Accurate 3D Shape Reconstruction and Generalizable 6D Force Estimation]] (2023): 继承 DTact 的灰度差深度重建，同时把硬件缩小，并用 dense deformation representation 学 6D force；适合看低成本结构如何扩展到力估计。
- [[@lu2024tacshade|TacShade: A New 3D-printed Soft Optical Tactile Sensor Based on Light, Shadow and Greyscale for Shape Reconstruction]] (2024): 从 TacTip 结构出发，用黑白图案的 light / shadow / greyscale 变化做粗 2.5D/3D reconstruction；适合看灰度边缘和 shape-from-shading 作为形状线索。
- [[@li2025r-tac0|R-Tac0: A Rounded High-Frequency Transferable Monochrome Vision-based Tactile Sensor for Shape Reconstruction]] (2025): 用黑白相机、白光和复合圆弧弹性体做 120 Hz 曲面触觉重建，并测试跨传感器迁移；适合看单通道、曲面、快速部署之间的取舍。

### 曲面 / 稠密形状 / 力与状态估计

这组文献更关心 curved / hemispherical / round（曲面、半球、圆形）形态，以及从内部图像回归稠密形状、力或完整接触状态。

- [[@do2022densetact|DenseTact: Optical Tactile Sensor for Dense Shape Reconstruction]] (2022): 半球形 soft elastomer 加鱼眼相机，用监督学习把 RGB 图像变成 radial depth image 和点云；适合看大接触面曲面传感器如何收集 3D 打印压头真值。
- [[@do2023densetact-2|DenseTact 2.0: Optical Tactile Sensor for Shape and Force Reconstruction]] (2023): 在 DenseTact 1.0 基础上缩小硬件、加入随机纹理和 6-axis wrench estimation；适合看同一内部图像同时做 shape reconstruction 与 net force / torque。
- [[@azulay2024allsight|AllSight: A Low-Cost and High-Resolution Round Tactile Sensor with Zero-Shot Learning Capability]] (2024): 3D 打印圆形光学触觉传感器，估计 3D contact position、3D force 和 torsion，并验证 zero-shot / few-shot transfer；适合看圆形大接触面和完整接触状态估计。

### 标定 / 制造 / 设计优化

这组文献解决的是“怎么更低成本、更可复现地得到可用传感器”，而不是单纯提出一个新的重建网络。

- [[@kota2026-3dcal|3D Cal: An Open-Source Software Library for Calibrating Tactile Sensors]] (2026): 把低成本 FDM 3D printer 改成自动 probing device，为 DIGIT / GelSight Mini 采集标定数据，并用 TouchNet 从 RGB + xy 预测 gradients；适合看自动标定、坐标通道和少量采样消融。
- [[@fan2025crystaltac|CrystalTac: Vision-Based Tactile Sensor Family Fabricated via Rapid Monolithic Manufacturing]] (2025): 用多材料 3D printing 一体化制造不同机制的 VBTS family，强调 fabrication consistency 和批量成本；适合看制造流程如何影响传感器族的复现性。
- [[@agarwal2025pbr-design|Vision-based tactile sensor design using physically based rendering]] (2025): 用 physically based rendering 模拟光源、几何、厚度和涂层材料，再用 RGB2Normal objective 评价设计；适合看 trial-and-error 的光学结构设计如何转为仿真优化。

### 多照明 / 边缘线索 / 图像融合

这条线不一定都来自触觉传感器，但都在回答“改变照明或融合多帧能不能得到更清楚的形状/边缘线索”。

- [[@raskar2004non-photorealistic-camera|Non-Photorealistic Camera: Depth Edge Detection and Stylized Rendering Using Multi-Flash Imaging]] (2004): 用多个 flash 在 depth discontinuity 旁制造 shadow sliver，通过 ratio image 检测 depth edges；适合借用“多照明直接产生几何边界 primitive”的思想。
- [[@redkin2024dynamic-illumination|Enhance Vision-based Tactile Sensors via Dynamic Illumination and Image Fusion]] (2024): 在 DIGIT 上切换 RGB LED patterns 并融合多帧，提升 contrast、sharpness 和 background difference；适合看动态照明带来的质量提升与延迟代价。
